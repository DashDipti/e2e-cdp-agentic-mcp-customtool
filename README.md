# e2e-cdp-agentic-mcp-customtool
Simple end to end demo showcasing Cloudera's Agentic AI capabilities leveraging Cloudera Platform components for Data In Motion, Data Warehouse, Data Engineering, MCP Server and Custom tool to showcase how easy it is to develop without worrying about the integration of all the various components of the data lifecyle.
